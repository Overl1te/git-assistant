SSH с Windows PC

              
Если проект remote и статус «ошибка / timed out» — проблема почти всегда на стороне сети или OpenSSH на Windows, а не в UI.

              
    
                
  1. На Windows: Get-Service sshd — статус Running. Иначе: Settings → Optional Features → OpenSSH Server.
    2. 
                
  2. Firewall: разрешить входящий TCP 22 (правило OpenSSH SSH Server).
    3. 
                
  3. С сервера Git Assistant: ssh -v -o ConnectTimeout=5 user@IP
    4. 
                
  4. В проекте: shell = powershell, path как на Windows, host = user@IP.
    5. 
                
  5. В карточке проекта нажми «Проверить SSH».
    6.
